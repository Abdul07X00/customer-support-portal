## Steps to run the project:

- Run `git clone https://github.com/ishancs/customer-support-portal.git`
- `composer install`
- `npm install`
- `php artisan serve`
- `npm run dev`
- Create sql database with name `tickets`
- `php artisan migrate:install`
- `php artisan db:seed`
- Open `http://localhost:8000/` in the browser

- Chat List
